# نظام إدارة العملاء - شركة الأمين

## التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
npm install
```

### 2. تشغيل الخادم
```bash
npm start
```

أو للتطوير مع إعادة التشغيل التلقائي:
```bash
npm run dev
```

### 3. الوصول للنظام
افتح المتصفح على: `http://localhost:3000`
